# Development

Development documentation is the reference for contributors building OrqaStudio. It covers the coding standards that all implementation must meet, the commands used to build, test, and check the codebase, and the setup steps for a new development environment.

Pages here include: coding standards, commands reference, setup guide, and the agentic workflow documentation.
