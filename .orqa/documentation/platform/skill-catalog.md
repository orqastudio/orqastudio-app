# Core Platform Knowledge Catalog

These knowledge artifacts ship with the OrqaStudio platform and are available to all agents. Each knowledge artifact provides domain-specific instructions that agents load on demand.

## Governance & Process

| Skill | Description |
|---|---|
| `planning` | Documentation-first workflow, Systems Architecture Checklist, UX-first design |
| `orqa-governance` | Governance patterns, rule enforcement, artifact lifecycle |
| `governance-maintenance` | How to maintain and evolve governance artifacts |
| `schema-validation` | Platform schema validation rules and patterns |
| `artifact-relationships` | Canonical relationship vocabulary with natural-language sentences |
| `artifact-status-management` | Status transitions, lifecycle gates, auto-transitions |
| `orqa-artifact-audit` | Methodology for auditing artifact graph integrity |
| `naming-conventions` | Naming standards for artifacts, files, and identifiers |

## Research & Evaluation

| Skill | Description |
|---|---|
| `architectural-evaluation` | How to evaluate architectural compliance and design options |
| `research-methodology` | Structured research process: scope, gather, analyse, present |
| `systems-thinking` | Holistic analysis, dependency mapping, second-order effects |
| `diagnostic-methodology` | Root cause analysis, hypothesis testing, debugging workflow |
| `restructuring-methodology` | Safe refactoring patterns, incremental migration |
| `composability` | Component composition, module boundaries, interface design |
| `tech-debt-management` | Tech debt tracking, prioritisation, and resolution |
